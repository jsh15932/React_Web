# Web Project with ReactJS

## Features
- 회원가입 / 로그인
- 메모 작성, 관리 (수정, 삭제, 추천)
- 회원 검색

## Following Technologies
- axios
- babel
- express
- bcryptjs
- mongoose
- react
- react-router
- react-addons-css-transition-group
- react-addons-update
- redux
- redux-thunk
- materializecss
- react-hot-loader
- webpack
- webpack-dev-server
- style-loader
- css-loader

## Scripts

- `npm run clean` 빌드 파일 삭제
- `npm run build` 빌드
- `npm run start` 서버 (port 3000)
- `npm run development` 개발 서버 (port 4000)
